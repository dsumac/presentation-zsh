Completion

Out-of the box

  • Out-of the box != bash_completion
  • avec cd sans chemin: sous bash cela affiche tous les fichiers ! et répertoires. Zsh uniquement les répertoires
  • avec kill. Bash affiche le numéro des processus + nom des . Zsh propose le parcours de la liste avec les info
  • avec git. Idem + sous commandes / options
  • avec netstat

Path expansion

Magic TAB

  • Complétion chemin unique: [cd ou sans cd] /u/lo/b: bash ne fait rien. zsh complète
  • Complétion chemin ambigue: /u/l/b. bash rien. zsh propose (cycling through) le chemin sur les répertoires non uniques

Path replacement

cd error lib

  • cd /usr/include/toto. Erreur: on souhaite /usr/include/toto => cd toto titi
  • Idem avec completion
  • Idem au milieu du chemin: ex fonctionnel /opt/workspaces/front-

Globbing / recherche

Recursive

=! sous répertoires

  • Dans /var
  • bash: ls -al **/*.log. juste les fichiers du répertoire demandé => premier niveau
  • zsh: ls -al **/*.log. Tous les fichiers de logs du répertoire et des ses sous-répertoires

Alias

Classique

Suffix: -s

Global: -g

  • Alias normal: alias l='ls -lah'
  • Suffix: Open with. Alias -s
  • alias -s html=geany
  • Global Alias -g:
  • Dans le rep de la pres: find . -name '*.html' | xargs grep 'expansion'
  • alias -g xg="| xargs grep"
  • find . -name '*.html' xg 'expansion'
